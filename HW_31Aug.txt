Homework (31August)

1. Как скопировать на свой компьютер репозиторий из github.com?
git clone и ссылка SSH c github.com - клонируем репозиторий
(предварительно создаем необходимые файлы)
git add . - добавляем все файлы
git commit -m "комментарий" - комментируем, закомитили
git push - отправляем на github.com

2. Как создать репозиторий из локального каталога на своем компьютере?
git init - создаем пустой репозиторий
git add - добавляем название необходимого файла
git commit -m "комментарий" - зафиксировали
Далее создаем на github.com новый репозиторий и выполняем все рекомендованные команды по инструкции
git remote -v - определяем связан ли с github.com.
Появится две строки origin.....(fetch), origin......(push)
git branch -M main - переименовываем если необходимо с master в main
git push - отправили на github.com

3. Как получить на локальный компьютер из менения с github.com?
git pull - вытягиваем все изменения с github.com

4. Какую информацию дают команды:
git status - показывает статус файлов. (Если не загружен, то файл красного цвета. Когда файл добавлен - он зеленый и написано add File)

git log - показывает что и кем было сделано: Author, Date, Add File и т.д.

git remote -v -определяет и показывает нам , связан ли даннвый файл с репозиторием github.com и предлагает нам его запушить на репозиторий
